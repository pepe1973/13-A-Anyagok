Webszerver kiépítése és elindítása

server.js:

Nyissunk meg a Visual Studio-t és szerkesszük a server.js állományt.

Vagy a kommentek nélkül.

Szerver elindítása:

Nyissunk meg a Git Bash-t és üssük be a következőt.

npm run dev

Látható, hogy a szerver elkezdte figyelni a 3500 portszámot lekérdezésekre (request) várva.

A böngészőben ez látható.
