npm csomagok beállítása

Nyissunk meg a parancssort és üssük be a következőt.

npm --version

Ha a Node.js telepítve van, akkor az npm csomagkezelő is. Ha úgy érezzük régi a verziószám, akkor frissítsük. Ilyenkor irány a honlap.

Ha nincs telepítve, akkor főleg irány a honlap.

package.json:

Hozzunk létre egy mappát backend néven.

Ha minden rendben akkor adjuk ki a következő utasítást.

npm init -y

Szükséges npm csomagok installálása:

Telepítsük a következő npm csomagokat.

npm install dotenv express ejs

Majd a következő npm csomagot csak fejlesztéshez.

npm install -D nodemon

Ekkor a következő dolgok jönnek létre:

  • node_modules - npm csomagok tárolója
  • package-lock.json - függőségek listázása

Egyiket se szerkesszük, ha nem tudjuk mit csinálunk!
